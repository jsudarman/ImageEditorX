\subsection{Detailed Description}
\hyperlink{interface_line_item_type}{LineItemType}. Contains the values that are common to the itemized order information of a object. NOTE: Please check AIM API XML Guide for valid values for each field. All fields are required. Authorize.Net Gateway will return an error if any of the field is missing.
